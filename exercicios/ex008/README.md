📝 Resumo do Projeto

Explorei as diversas formas de formatação de texto no HTML5, focando especialmente na diferença entre tags puramente visuais e tags semânticas, que agregam significado e acessibilidade ao conteúdo. Pratiquei a implementação de elementos essenciais como negrito, itálico, textos marcados com estilo personalizado via CSS, além de representações específicas para fórmulas químicas e equações matemáticas. 
Então criei uma página para testar e comparar as diversas tags de formatação. O foco não foi apenas deixar o texto "bonito", mas entender a semântica web:
1. Diferenciando Visual de Significado
2. Marcações Especiais
3. Simulação de Edição de Documentos
4. Textos Técnicos

Detalhes Valioso que Aprendi:
- Acessibilidade importa: tags semânticas ajudam pessoas que usam leitores de tela a entenderem melhor o conteúdo.
- Organização: Usei títulos hierárquicos para que a página ficasse bem estruturada e fácil de ler
